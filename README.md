Tracalorie is a full-stack web application built with JavaScript and HTML that allows users to track calories, manage their daily nutrition intake, and gain insights into meals and dietary habits. The backend logic is implemented in app.js, while the front-end provides a clean interface via index.html. Users can add, edit, and log meals; view nutrition summaries; and monitor progress over time. Technologies include JavaScript (ES6+), HTML5, and a modular architecture enabling maintainability. Suitable for personal health tracking or as a foundation for more advanced diet-management systems.
